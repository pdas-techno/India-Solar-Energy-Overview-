# ☀️ India Solar Energy Analysis Dashboard

## 📌 Project Overview
The India Solar Energy Analysis Dashboard is a Power BI and SQL-based data analytics project focused on analyzing solar energy production across different states of India.

This project provides detailed insights into:
- Solar Power Generation
- Installed Capacity
- CO₂ Emissions Saved
- Maintenance Costs
- Monthly Energy Trends
- State-wise Performance Analysis

The dashboard was created using a clean CSV dataset, SQL queries, DAX measures, and interactive Power BI visualizations.

# 🚀 Technologies Used

- Power BI

- CSV Dataset

# 📂 Dataset Information

The dataset contains:
- Date
- State
- Sunlight Hours
- Temperature
- Installed Capacity (MW)
- Power Generated (MWh)
- CO₂ Saved (Tons)
- Maintenance Cost (INR)

# 📈 Key Insights

- Rajasthan and Gujarat showed high solar energy generation.
- Increased sunlight hours positively impacted power generation.
- CO₂ emissions savings increased with installed solar capacity.
- Monthly trends helped identify peak solar production periods.
